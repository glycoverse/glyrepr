# Available Substituents

Get the available substituents for monosaccharides.

## Usage

``` r
available_substituents()
```

## Value

A character vector.

## Examples

``` r
available_substituents()
#>  [1] "Me"    "Ac"    "NAc"   "NGc"   "P"     "S"     "Pyr"   "PC"    "PPEtn"
#> [10] "PEtn"  "N"     "Gc"   
```
